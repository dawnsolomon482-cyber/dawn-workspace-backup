## How to Think About Any SEO Problem

Before answering, identify which layer the problem is at:

1. **Entity layer** — Does Google know and trust this business? (GBP, schema, sameAs, NAP)
2. **Technical layer** — Can Google crawl and index the site properly? (speed, sitemap, structure)
3. **Content layer** — Is there enough topical authority to rank? (keywords, clusters, briefs)
4. **Authority layer** — External proof of credibility? (backlinks, citations, press, reviews)
5. **GEO layer** — Is the entity optimized for AI discovery? (AI Overviews, ChatGPT, Perplexity)

Fix in this order. A site with no entity signals cannot rank even with great content.

## Full List of SEO Tasks (All Services)

### 1. SEO Audit
- Technical site audit (speed, indexing, crawl errors, duplicate content)
- On-page audit (title tags, H1s, meta descriptions, URL structure)
- Content audit (thin content, missing keywords, outdated posts)
- Entity audit (Brand SERP check, schema presence, NAP consistency)
- GBP audit (completeness, category, photos, reviews, posts)
- Backlink audit (toxic links, link profile quality)
- Competitor gap analysis (keywords they rank for that client doesn't)

### 2. Google Business Profile Management
- Claim and verify GBP listing
- Complete all profile sections (name, category, description, services, attributes)
- Upload and optimize photos (logo, cover, interior, exterior, team, work)
- Seed Q&A section with common questions
- Set up and publish weekly GBP posts
- Respond to ALL reviews (positive and negative)
- Monitor and flag spam/fake reviews
- Update holiday hours and special announcements
- Monitor GBP Insights monthly (views, calls, direction requests, clicks)
- Manage suggested edits from Google

### 3. Citation Building (US)
- Build Tier 1 citations: GBP, Apple Maps, Bing Places, Yelp, Facebook, LinkedIn, BBB
- Build Tier 2 citations: Chamber of Commerce, Manta, Hotfrog, Nextdoor, Alignable
- Build industry-specific directory listings
- Fix inconsistent NAP across existing citations
- Monitor and update citations when business info changes
- Target: 25-40 consistent citations for competitive US markets

### 4. Entity Building
- Create/optimize About page as Entity Home
- Add Organization or LocalBusiness JSON-LD schema with sameAs on homepage
- Add Founder/Author pages with bio, credentials, photo
- Create Wikidata entry (if none exists)
- Link all social profiles consistently
- Ensure NAP in site footer matches GBP exactly

### 5. Technical SEO
- Fix Google Search Console errors (indexing, coverage, Core Web Vitals)
- Submit and verify XML sitemap
- Configure robots.txt properly
- Set correct permalink structure (/%postname%/)
- Fix HTTPS issues and mixed content
- Optimize page speed (caching, image compression, CDN)
- Fix duplicate content (canonical tags, noindex for thin pages)
- Ensure mobile-friendly design
- Implement breadcrumbs

### 6. On-Page Optimization
- Optimize title tags (keyword near front, under 60 characters)
- Write meta descriptions (120-155 characters, include CTA)
- Ensure one H1 per page with target keyword
- Fix header hierarchy (H1 > H2 > H3)
- Optimize URL slugs (short, keyword-focused, no dates)
- Add target keyword in first paragraph
- Optimize image filenames and alt text
- Add internal links (3-5 per page, descriptive anchor text)
- Ensure keyword density is natural (match competitor density)

### 7. Schema Markup
- Organization / LocalBusiness schema on homepage (with sameAs)
- Article schema on all blog posts
- FAQ schema on service pages and FAQ sections
- HowTo schema on tutorial/guide content
- Product schema on product/service pages
- Review schema where applicable
- Breadcrumb schema
- Validate all schema via Google Rich Results Test

### 8. Keyword Research
- Identify primary target keywords (BOFU first)
- Find low-competition, high-intent keywords (Avalanche Method)
- Map keywords to existing pages
- Identify content gap vs competitors
- Build keyword clusters (pillar + supporting topics)
- Prioritize: High intent + Low KD + Decent volume = do first

### 9. Content Strategy & Creation
- Create content plan (pillar page + 5-10 supporting articles per topic)
- Write SEO content briefs (keyword, intent, outline, semantic subtopics, word count)
- Write or edit blog posts to target keyword clusters
- Create location pages for each city served
- Optimize existing content (refresh titles, update stats, add keywords)
- Find pages ranking #10-50 and upgrade them (fastest wins)
- Write FAQ content for AI Overview optimization

### 10. Local SEO (USA-Specific)
- Optimize for local keyword format: [service] + [city, state]
- Create and optimize location pages per city served
- Build local backlinks (Chamber, local news, sponsor events)
- Set up and manage Nextdoor business profile
- Optimize for "near me" searches
- Track Map Pack rankings (LocalFalcon or BrightLocal)

### 11. Link Building
- Local link building (Chamber of Commerce, local news, event sponsorships)
- HARO / Connectively — respond to journalist queries for press mentions
- Guest posting on industry blogs
- Partner with complementary local businesses for link exchanges
- Unlinked brand mention outreach (find mentions, ask for the link)

### 12. Review Strategy
- Set up review request process (text or email after service)
- Create direct review link (GBP share link)
- Train client on when and how to ask for reviews
- Monitor all review platforms monthly
- Respond to all reviews within 48 hours
- Flag and dispute fake/spam reviews

### 13. GEO — AI Optimization
- Ensure entity is established in Google's Knowledge Graph
- Optimize key pages for featured snippets (answer in first paragraph)
- Add FAQ sections to service pages
- Use structured content formats (numbered lists, tables, H2 questions)
- Create original data/statistics content (gets cited by AI)
- Monitor AI Overview appearances for target keywords

### 14. SEO Reporting (Monthly)
- GBP Insights report: views, calls, direction requests, clicks
- Ranking report: top 10 target keywords, Map Pack positions
- Google Search Console: impressions, clicks, average position, new indexing issues
- GA4: organic sessions, top landing pages, leads/conversions
- Citation growth: new citations added, total count
- Review growth: new reviews, rating trend
- Actions completed this month
- Actions planned next month

## How to Answer SEO Questions

**"Where do we start?"**
→ Audit first. Search the brand name, check GBP, run PageSpeed, check GSC errors.
→ Fix entity and technical first. Then keyword research, then content.

**"How do we rank in the Map Pack?"**
→ Verify and complete GBP. Build Tier 1 citations. Get consistent reviews. Add LocalBusiness schema.

**"How do we rank faster?"**
→ Pages ranking #10-50 are fastest wins — upgrade those first.
→ Fix entity signals — fastest Google trust boost.

**"No rankings at all?"**
→ Entity not established. Start: GBP, About page, Organization schema, Tier 1 citations.

**"Client wants more Google reviews"**
→ Create direct review link. Set up text follow-up sequence post-service. Aim for 2-4/month consistently.

**"How do we show in AI answers?"**
→ Entity must be recognized first. Then: structured data + direct answers in first paragraph + FAQ sections.
