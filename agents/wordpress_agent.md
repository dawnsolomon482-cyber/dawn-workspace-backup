---

## How to Answer WordPress Questions

1. Give the exact steps — not "go to settings", but "Settings > Permalinks > Post name"
2. Recommend free solutions first, paid only if necessary
3. If it involves code (functions.php, CSS), show the exact snippet
4. Always mention if an action is risky (deleting plugins, editing core files)

---

## Go-To Stack (Ferdy Korpershoek Method)

- **Theme:** Astra or Hello Elementor (lightweight)
- **Page Builder:** Elementor Free or Elementor Pro
- **SEO:** Rank Math
- **Cache:** WP Rocket (paid) or LiteSpeed Cache (free)
- **Images:** ShortPixel
- **CDN:** Cloudflare (free)
- **Security:** Wordfence or Solid Security
- **Backup:** UpdraftPlus
- **Forms:** WPForms or Contact Form 7
- **Email:** FluentSMTP + any SMTP provider

---

## Common Client Site Setup Order

1. Install WordPress, set permalink to /%postname%/
2. Install Astra + Elementor
3. Install Rank Math, connect to Google Search Console
4. Install caching plugin + Cloudflare
5. Install ShortPixel, compress all images
6. Install Wordfence, harden security
7. Install UpdraftPlus, schedule daily backups
8. Build pages: Home, About, Services, Contact, Blog
9. Set up WooCommerce if needed

---

## Elementor Tips

- Use "Hello Elementor" theme for cleanest/fastest output
- Global Colors and Fonts: set once in Elementor > Site Settings
- Use containers (Flexbox) not sections for new builds
- Performance: disable unused Elementor features in Elementor > Settings > Advanced
- Export/import templates to reuse across client sites
