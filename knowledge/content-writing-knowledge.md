# Content Writing Knowledge Base (SEO-Focused)

## Core Principle
Write for humans first, search engines second.
Content that ranks = content that fully answers the searcher's intent.

---

## Content Types That Rank

1. **Informational** — "How to...", "What is...", "Best way to..."
2. **Comparison** — "X vs Y", "Best tools for..."
3. **Listicles** — "10 ways to...", "Top 5..."
4. **Local** — "[Service] in [City]"
5. **Product/Service pages** — target buyer-intent keywords

---

## Blog Post Structure (HeyTony Method)

```
H1: Primary Keyword (Title)

Intro (100-150 words):
- Hook: state the problem
- Promise: what they'll learn
- No fluff

H2: Section 1 (secondary keyword)
- 150-300 words
- Answer directly, then expand

H2: Section 2
...

H2: FAQ Section (use FAQ schema)
- 3-5 questions people actually search
- Short, direct answers (2-3 sentences)

Conclusion (50-100 words):
- Summarize key points
- One CTA (call to action)
```

---

## Keyword Targeting Per Post

- 1 primary keyword (H1, first 100 words, URL, meta title)
- 2-3 secondary keywords (H2s, body text)
- LSI keywords (related terms — use naturally)
- Target keyword difficulty under 30 for new sites

---

## Content Length Guide

| Type | Word Count |
|------|-----------|
| Short informational | 800-1200 |
| Standard blog post | 1200-2000 |
| Pillar/cornerstone content | 2500-4000 |
| Product page | 300-600 |
| Local landing page | 500-800 |

---

## AI-Assisted Content Workflow

1. **Research** — find top 3 ranking articles for the keyword
2. **Outline** — create H2/H3 structure based on what all top pages cover
3. **Draft with AI** — use Claude to write section by section
4. **Humanize** — add personal examples, local context, unique insights
5. **Optimize** — add keyword in title, H1, first paragraph, meta description
6. **Add FAQ schema** — via Rank Math
7. **Internal link** — link to 3 related posts
8. **Publish + submit URL** to Google Search Console for indexing

---

## Writing Rules

- Short sentences. Max 20 words per sentence.
- Short paragraphs. Max 3-4 lines.
- Use bullet points and numbered lists for steps
- Bold key terms and important points
- No keyword stuffing — mention primary keyword 3-5x per 1000 words
- Always answer the question in the first 2 sentences of the intro

---

## Meta Title Formula

- `Primary Keyword - Brand Name` (max 60 chars)
- Or: `Primary Keyword: What You Need to Know`
- Include year if content is time-sensitive: `Best WordPress Plugins 2025`

## Meta Description Formula

- Max 160 chars
- Include primary keyword
- End with a call to action
- Example: "Looking for the best WordPress SEO plugins? Here are the top 5 tools to rank faster in 2025. See full list."
